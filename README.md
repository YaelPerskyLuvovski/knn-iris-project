# KNN Iris Project

## Description
This project implements the K-Nearest Neighbors (KNN) algorithm on the Iris dataset.

## Steps
- Load the dataset
- Split data into training and testing sets
- Train the KNN model
- Evaluate accuracy
- Plot confusion matrix
- Plot accuracy vs K
